Not every export needs to land in Lacerte, CCH Axcess, or GoSystem. When you want the approved data as a standalone file — for a workpaper, a partner review, a client handoff, or a spot-check by a senior preparer — Byron offers two download destinations on the Create export dialog: **Excel** and **PDF**.

The Approved status requirement from [Export overview](/exporting/overview) applies to both.

## Excel export

The Excel workbook is the more substantive of the two download options. It captures the full extracted data in a structured workpaper that you can analyze, share, or use as a permanent record of what Byron extracted from the source documents.

### When to use Excel

Use Excel when:

- You use tax software other than CCH Axcess, Lacerte, or GoSystem.
- You want to perform additional analysis or calculations before tax entry.
- You need a client-facing summary of all tax documents.
- You're preparing returns manually or using a custom workflow.
- You want a permanent reference copy of extracted data.

### What's in the workbook

The workbook contains each document in the export alongside the detailed fields Byron extracted from it. It's generated dynamically from whatever documents you include in the batch — export a single K-1 and the workbook covers that K-1; export a full client package and the workbook covers the package.

### Basic Excel workpaper

All non-K-1 documents — W-2s, 1099s, 1098s, and so on — are exported into the Excel file as a summary page plus a detailed tab for each document.

### The K-1 aggregated view

Byron's Excel workbook has a specific view built for K-1s: a single sheet that aggregates values across every K-1 in the batch.

- **K-1 (1065) Workpaper** — Partnership K-1s and associated state K-1s.
- **K-1 (1120S) Workpaper** — S-Corporation K-1s and associated state K-1s.
- **K-1 (1041) Workpaper** — _Coming soon._

<Warning>
  Changes made in the Excel file are not synced back to Byron. The Excel export is a point-in-time snapshot.
</Warning>

<Note>
  **Need a different Excel format?** Byron is happy to work with firms that have a specific Excel template they'd like the export to match. If your firm has a preferred workpaper format you'd like us to support, reach out to [support@usebyron.com](mailto:support@usebyron.com).
</Note>

## PDF export

Byron currently exports a **collated PDF** — every source document in the batch combined into a single download. It's useful for client handoffs, partner review packages, or a single-file record of the source documents that fed a filing.

**Bookmarked PDF support is in development.** Once it lands, exported PDFs will include navigable bookmarks so reviewers can jump straight to a specific document inside the file.
