title: "Supported tax document types"
---

Team at Byron is working hard to make sure we can support you with as many types of tax documents as possible. 

We currently fully support the following:

| ✓ W-2 | ✓ SSA-1099 | ✓ Schedule C |
| :-- | :-- | :-- |
| ✓ 1099-INT | ✓ 1098 | ✓ Schedule E |
| ✓ 1099-DIV | ✓ 1098-T | ✓ Schedule F |
| ✓ 1099-MISC | ✓ 1098-E | ✓ 1098-Q |
| ✓ 1099-NEC | ✓ 1095-A | ✓ K-1 (Form 1065) - Partnership |
| ✓ 1099-NEC | ✓ 1095-A | ✓ K-1 (Form 1120S) - S-Corporation |
| ✓ 1099-R | ✓ 1040-ES | ✓ K-1 (Form 1041) - Estate/Trust |
| ✓ 1099-B | ✓ Form 4868 | ✓ K-3 (Form 1065) - International Partnership |
| ✓ 1099-SA | ✓ Form 2350 | ✓ 55 State K-1s |

<Tip>
  You need a different form?

  Our technology uses AI to build out form support at a rapid pace. If you have any additional forms for which you need support, provide us sample data and we can build it!
</Tip>
